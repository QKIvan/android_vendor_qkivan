# vendor_addons

## How to use

### 1.Clone this repo

```
git clone https://github.com/QKIvan-Projects/vendor_addons addons -b main
```

### 2.Inherit this repo

Add

```
$(call inherit-product-if-exist, vendor/addons/config.mk)
```

in your device tree.
